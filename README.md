# Windows Script Kit
A collection of practical batch and PowerShell scripts for automating common Windows tasks, fixes, and hidden tweaks.

## Features
- Generate battery health reports
- Run system file checks and repairs (SFC)
- Disable Start menu web search
- Automate fresh Windows setup
- More utilities coming over time

## Usage

Run scripts individually depending on your needs.
Some scripts may require administrator privileges.

## Disclaimer
These scripts are designed for Windows 10 Pro and 11 Home.
Review scripts before running to understand what changes are being made.
Use at your own risk. Always back up important data before making system changes.
